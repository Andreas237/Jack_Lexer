## System Requirements
* Unix based OS
* JRE 1.5+
* Tested on OSX 10.13.4



## File


### Input Jack Files
These can be anywhere in the PJ03 Directory, must end in *_.jack_* and must not have 
*test* in the name.

### Out Files
*_.tok_* and *_.log_* files will be generated in *_PJ03/results/_*
